# Narrate — The N Layer

This folder is the narrative layer of Lincoln Mitchell's INC framework (Ideate < Narrate > Create).

It contains structured context that agents and humans can consume — not polished content, but working knowledge. The raw material that sits between ideation and creation.

## ids-2026/

Structured analysis of all 16 talks from [Into Design Systems 2026](https://intodesignsystems.com/) (March 19-20, 2026). Each file contains:

- Speaker, company, talk title, links
- Key concepts and architecture
- Audience Q&As (extracted from FigJam via Figma MCP)
- Lincoln's answers (from the Room After the Talk)
- Positioning analysis (Lincoln's methodology vs speaker's approach)
- Content opportunities identified

These files were generated through a combination of:
- Live conference note-taking in FigJam
- Automated extraction via Figma MCP → Claude Code
- Analysis and positioning by the agent team defined in CLAUDE.md

This folder practices what it describes: machine-readable context that any agent can consume to understand Lincoln's position in the agentic design systems landscape.

**Everyone contributes to N. Anyone can draw from it.**
