Designed 2026-03-30. This is the living UX spec for the intent-based UI demo page.

**Why:** /explore is the proof-of-practice for INC: I = user types intent, <N> = keyword routing layer selects a surface, C = a structured visual surface renders from the design system. Replaces the chatbot/avatar concept.

**How to apply:** FED Developer builds against this spec. UI Designer applies visual treatment (tokens, motion curves) to the structural layout described here.
