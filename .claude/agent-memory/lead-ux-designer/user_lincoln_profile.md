Lincoln Mitchell is a Design Systems Architect (50% Designer / 30% FED / 40% PO), founder of Interfaces-N-Creatives. He is a genuine hybrid — not a designer who learned to code. Based in London and Sydney.

The portfolio site (Next.js App Router, Tailwind v4, dark editorial aesthetic) is a proof-of-practice: the site demonstrates agentic design systems by being built with them.

**INC Framework (core mental model):**
- I = Ideate / Intent (human spark, right hemisphere, NorthStar)
- <N> = Narrate (agentic layer, design system as context infrastructure, encodes domain knowledge)
- C = Create / Directive (execution, left hemisphere, production output)

**Recent insight (2026-03-30):** I = Intent-Based UI, N = Agentic routing layer, C = Directive UI output. The /explore page is the proof of this mapping.

**Design tokens:** Plus Jakarta Sans (display), Inter (body), Space Grotesk (mono). Dark surfaces: --color-surface #0b1326. Primary: #b4c5ff (electric blue). Secondary: #d0bcff (purple). Tertiary: #4cd7f6 (cyan). Warm: #f59e0b (amber).

**Key collaborators:** UI Designer (tokens/look), FED Developer (React/TypeScript implementation), UX Researcher (domain/user needs), Lincoln as DS Architect and final decision-maker.
